# SQL Utilities

Scripts operacionais de apoio que nao fazem parte do pipeline de migrations do Flyway.

- `ADD_CUSTOMER_TO_USER_PROFILES.sql`: ajuste manual pontual de schema.
- `CLEAN_AND_RESTART.sql`: reset controlado de ambiente de desenvolvimento.
- `check-migrations.sql`: verificacao manual de estado das migrations.

Use esses arquivos apenas em manutencao assistida.
